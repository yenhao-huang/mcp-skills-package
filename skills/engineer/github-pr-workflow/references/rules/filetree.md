# Filetree Rules

- Read contribution rules and PR templates from the target repository before
  drafting the pull request.
- Stage and commit only files belonging to the requested PR.
- Preserve unrelated working-tree and untracked files.
- Keep temporary PR body files outside the repository or remove them before
  completion.
- Never include `.env`, credentials, private keys, generated secrets, or
  unredacted sensitive logs in commits or PR text.
